# native_rbf

NativeRBF - система pose-space деформации (аналог RBF-солвера) для Autodesk
Maуа, построенная целиком на нативных нодах - без С++-плагина. Все готовые
RBF-решения - это компилируемые плагины,
которые фрилансеры-аниматоры часто не могут или не хотят устанавливать, а
встроенный Pose Interpolator не всегда способен покрывать поставленные
задачи. Сцена с NativeRBF работает у
любого получателя без установки чего-либо; Python-редактор нужен только на этапе
сетапа, не во время работы сцены.

Система разделена на две независимые части: Солвер - самодостаточный нодграф в
сцене, ничего не знающий о редакторе, и Редактор - «компилятор», который
строит/перестраивает/чинит этот граф по декларативным данным на МЕТА-ноде
